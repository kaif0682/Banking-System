# 🏦 Banking System Project

## 📌 Overview
This is a Banking System project developed to simulate basic banking operations like account creation, deposit, withdrawal, and balance checking.

---

## 🚀 Features
- Create new bank account
- Deposit money
- Withdraw money
- Check account balance


---

## 🛠️ Technologies Used
- Programming Language: (Python /Django /HTML /CSS )
- Tools: Git, VS Code
